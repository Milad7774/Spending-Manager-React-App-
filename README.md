#  Spending Manager - React Expense Manager

A clean, responsive expense tracking application built with React. Keep track of your purchases, organize them by date, and see your total spending at a glance. Perfect for personal budgeting.

##  Live Demo
[View Live Demo](https://milad7774.github.io/Spending-Manager-React-App-/)

## Features

### Core Features
-  Add purchases with item name, price, and date
-  View all purchases in a sortable table
-  Delete individual purchases
-  Automatic total calculation
-  Local storage persistence (data survives page refresh)
-  Responsive design (works on mobile, tablet, desktop)

### UX Enhancements
-  Price formatting with commas (e.g., `1,000` instead of `1000`)
-  Date picker for easy date selection
-  Clean, modern UI with gradient effects
-  Toast notifications for success/error feedback

## Technologies Used

- **React 18** - UI library
- **React Router DOM** - Navigation between pages
- **React Hot Toast** - Toast notifications
- **CSS3** - Styling with Flexbox/Grid
- **LocalStorage API** - Data persistence
